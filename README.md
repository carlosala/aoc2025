# ADVENT OF CODE 2025

These are my personal solutions to the [Advent of Code 2025](https://adventofcode.com/2025) written in Rust. I
will try to post each solution the same day that it comes out, but I don't promise anything!

**NOTE:**

- _You have to add a `.env` file in the root project with the following values in order to use the `cargo run -- download`._

```sh
  TOKEN="your_session_cookie_from_aoc"
```
